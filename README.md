# Rock, Paper, Scissors Game 🎮

A simple, interactive Rock, Paper, Scissors game built using web fundamentals. This project was developed as a practical exercise to reinforce fundamental front-end development concepts, improve logic building, and master UI/UX styling.

## 🚀 Learning Objectives
* **HTML5:** Structuring clean, semantic web pages.
* **CSS3:** Implementing responsive layouts, custom styling, and smooth transitions.
* **JavaScript (ES6+):** Mastering DOM manipulation, event listeners, conditional logic, and state management.

## ✨ Features
* Responsive design that works on desktop and mobile screens.
* Live score tracking for both the player and the computer.
* Interactive UI with visual feedback for wins, losses, and ties.
* Instant game reset functionality.

## 🛠️ Tech Stack
* HTML5
* CSS3
* JavaScript (Vanilla)
